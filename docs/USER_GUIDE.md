# User Guide - ImoAgent

## Getting Started

### First Login

1. Access the platform at your deployment URL
2. Sign up with your email or use social login
3. Complete your profile information
4. Choose your subscription plan

### Dashboard Overview

The dashboard is your command center with:
- **Statistics Cards**: Quick view of your key metrics
- **AI Agents Panel**: Status of all 7 AI agents
- **Quick Actions**: Fast access to common tasks
- **Features Overview**: Summary of available features

## Using AI Agents

### 1. Search Agent (Property Scraper)

Search across 7+ real estate portals simultaneously:

1. Go to **Search** section
2. Enter search criteria:
   - City/Location
   - Property type
   - Price range
   - Bedrooms/Bathrooms
3. Click **Search**
4. AI will scan all portals and aggregate results
5. View, filter, and save properties

**Portals Included**:
- ZAP Imóveis
- Viva Real
- Imovelweb
- OLX Imóveis
- QuintoAndar
- Chaves na Mão
- Tem Casa

### 2. Coaching Agent (SMART Goals)

Set and track SMART goals:

1. Go to **Goals** section
2. Create new goal with:
   - **S**pecific: Clear, defined objective
   - **M**easurable: Quantifiable metrics
   - **A**chievable: Realistic target
   - **R**elevant: Aligned with career
   - **T**ime-bound: Deadline
3. AI provides recommendations and tracking
4. Get weekly coaching insights

### 3. Gamification Agent

Compete and earn rewards:

1. View your **Ranking** on dashboard
2. Check **Achievements** earned
3. Play **Minigames** for bonus points
4. Track progress against peers
5. Unlock badges and rewards

**Ranking Periods**:
- Daily
- Weekly
- Monthly
- Yearly

### 4. Ads Agent

Optimize your property listings:

1. Select a property
2. Click **Optimize Ad**
3. AI analyzes and suggests:
   - Better titles
   - Enhanced descriptions
   - Optimal pricing
   - Best photos
4. Apply suggestions with one click

### 5. Legal Agent

Analyze documents and contracts:

1. Go to **Documents**
2. Upload contract or document
3. AI performs analysis:
   - Identifies key clauses
   - Highlights risks
   - Suggests modifications
   - Generates checklist
4. Download annotated version

### 6. Leads Agent

Automated lead management:

1. New leads are automatically scored (0-100)
2. AI routes to best available agent
3. Suggests follow-up actions
4. Tracks conversion probability
5. Recommends next steps

**Lead Scoring Factors**:
- Contact completeness
- Message quality
- Property interest
- Response time
- Historical behavior

### 7. Tracking Agent (Agenda AI)

Smart calendar and productivity:

1. **Calendar**:
   - Schedule appointments
   - Get AI suggestions for optimal times
   - Automatic reminders
   - Conflict detection

2. **Pomodoro Timer**:
   - 25-minute focus sessions
   - 5-minute breaks
   - Productivity tracking
   - Daily/weekly reports

## Property Management

### Adding Properties

1. Click **+ New Property**
2. Fill in details:
   - Title and description
   - Type (house, apartment, land, commercial)
   - Price and area
   - Bedrooms/bathrooms
   - Address and location
   - Photos (drag & drop)
3. Save and publish

### Managing Listings

- **Edit**: Update information anytime
- **Archive**: Remove from active listings
- **Share**: Generate shareable link
- **Analytics**: View performance metrics

## Lead Management

### New Lead Flow

1. Lead enters system (form, portal, manual)
2. AI scores the lead (0-100)
3. Lead routed to agent
4. Agent receives notification
5. Follow-up suggestions provided

### Lead Statuses

- **New**: Just received
- **Contacted**: Initial contact made
- **Qualified**: Meets criteria
- **Converted**: Became client
- **Lost**: Not converted

## Appointments & Calendar

### Scheduling

1. Go to **Calendar**
2. Click date/time slot
3. Enter details:
   - Title
   - Description
   - Duration
   - Location
   - Attendees
   - Property (if applicable)
4. Save appointment

### Pomodoro Sessions

1. Click **Pomodoro** timer
2. Set task description
3. Start 25-minute session
4. Take 5-minute break
5. Repeat as needed
6. Track productivity

## Troubleshooting

### Common Issues

**Problem**: AI agent not responding
- **Solution**: Check API keys in settings

**Problem**: Properties not syncing
- **Solution**: Verify Supabase connection

**Problem**: Payment failed
- **Solution**: Update payment method in settings

**Problem**: Can't upload documents
- **Solution**: Check file size (max 10MB)
